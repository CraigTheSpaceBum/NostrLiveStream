FAQ & Help
What is NIP-53 +
NIP-53 defines live streaming events on Nostr using kind:30311. Fill in the Go Live form and Sifaka Live broadcasts your stream to relays - any NIP-53 compatible client can discover it.
How do I self-host my stream +
Install nginx-rtmp on any VPS ($5/mo). Point OBS to your RTMP endpoint. Sifaka Live publishes the HLS URL to Nostr relays. No third-party CDN needed.
What is NIP-05 verification +
NIP-05 links your npub to a domain (e.g. craig@bitcoin-cats.ca). You host a JSON file at your domain. Verified users get a purple badge in chat and on profiles.
How do Zaps work (NIP-57) +
Add your Lightning address (lud16) to your NIP-01 profile. Viewers click Zap, their wallet pays a Lightning invoice, and the proof is published as a kind:9735 event in chat.
Which NIPs does Sifaka Live support +
NIP-01 (profiles), NIP-05 (verification), NIP-07 (browser signing), NIP-25 (reactions/likes), NIP-46 (remote signing), NIP-53 (live streams), NIP-57 (zaps), NIP-65 (relay lists).

FAQ and Help (Super Simple)

You are in the right place.

If this is your first time, read from top to bottom.

Short version: Sifaka Live helps people find live streams on Nostr.

What is Nostr?

Nostr is an open social network.

Open means no single company owns it.

You own your account -> Apps can change -> You keep your followers

What is Sifaka Live?

Sifaka Live is a stream directory.

Think: TV guide for Nostr live streams.

Important: Sifaka Live helps discovery. It does not host the video.

Why creators like this

  • Direct tips with zaps (Lightning payments)
  • No big platform middleman
  • You can run your own stream server
Money flow:

Old way: Viewer -> Platform -> Creator (fees taken)

Nostr way: Viewer -> Creator (direct)

Start streaming fast (Easy Way)

Best for beginners. No server setup needed.

1

Make a Nostr account

2

Install OBS

3

Open zap.stream

4

Paste Stream URL + Key in OBS

5

Click Start Streaming

Do this exactly:

  1. Open Primal (or Snort) and create your Nostr account.
  2. Install OBS Studio on your computer.
  3. Open zap.stream and sign in with your Nostr account.
  4. On zap.stream click Start Streaming and copy Stream URL + Stream Key.
  5. In OBS go to Settings -> Stream, paste URL + Key, then click Start Streaming.

Run your own server (Owncast)

This gives you the most control.

You control your stream, chat, and website.

1

2

3

4

5

Do this in order (click links below):

  1. Pick a server provider and launch a small VPS.
  2. Install Owncast using the official quickstart page.
  3. Open your Owncast /admin page and finish setup.
  4. Copy your Owncast stream key into OBS Settings -> Stream.
  5. Start streaming from OBS, then share your stream on Nostr.
