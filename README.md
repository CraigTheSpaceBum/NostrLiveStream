# Sifaka Live

**A directory for decentralized live streams on Nostr.**

Sifaka Live helps people discover live streams running across the Nostr
network.

Instead of relying on centralized platforms like Twitch or YouTube,
creators can run their own streaming servers and announce their streams
through Nostr. Sifaka Live collects those announcements and displays
them in a simple directory so viewers can find streams easily.

------------------------------------------------------------------------

# What Problem Does This Solve?

Traditional streaming platforms:

-   control creators' content
-   take a percentage of revenue
-   can ban or demonetize creators
-   own the audience relationship

Sifaka Live supports a different model.

Creators can:

-   run their **own streaming server**
-   **own their audience**
-   receive **direct Lightning payments (zaps)**
-   broadcast streams without a centralized platform

------------------------------------------------------------------------

# What is Nostr?

Nostr stands for **Notes and Other Stuff Transmitted by Relays**.

It is an open protocol that allows social apps to communicate over a
decentralized network.

Instead of accounts belonging to a company, users control their own
identity using cryptographic keys.

This means:

-   your identity works across many apps
-   no platform owns your account
-   anyone can build applications on the network

Sifaka Live uses Nostr to discover live streams across the ecosystem.

------------------------------------------------------------------------

# How Streaming Works

Traditional platforms:

Streamer → Platform → Viewers

Nostr streaming:

Streamer → Own Server → Nostr Network → Viewers

Sifaka Live acts as a **directory**, not a hosting platform.

It simply helps viewers discover streams.

------------------------------------------------------------------------

# Start Streaming in 5 Minutes

## Option 1 --- Easy Mode (Recommended)

Use a hosted Nostr streaming platform such as:

-   zap.stream
-   shosho.live

Steps:

1.  Create a Nostr account
2.  Install OBS Studio
3.  Log into a streaming host
4.  Copy your stream key
5.  Paste it into OBS
6.  Click **Start Streaming**

Your stream can now appear across Nostr apps and directories.

------------------------------------------------------------------------

## Option 2 --- Self Hosting (Advanced)

Run your own streaming server using **Owncast**.

Steps:

1.  Install Owncast on a server
2.  Open the Owncast dashboard
3.  Copy your stream key
4.  Paste the key into OBS
5.  Start streaming

You now control your own streaming infrastructure.

------------------------------------------------------------------------

# Zaps (Direct Creator Payments)

Nostr supports Lightning payments called **zaps**.

Viewers can send small payments directly to creators.

Traditional platforms:

Viewer → Platform → Creator (platform takes a cut)

Nostr:

Viewer → Creator

Payments are instant and go directly to the creator.
